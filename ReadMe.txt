By default, the application looks for the input file in the same directory.

Usage -

StoreExample.exe
(In no file specified, it looks for a file name input.txt in same directory)

StoreExample.exe example1.txt
(Looks for example1.txt in the same directory)

StoreExample.exe "Inputs\example1.txt"
(Looks for example1.txt in the "Inputs" subdirectory)
